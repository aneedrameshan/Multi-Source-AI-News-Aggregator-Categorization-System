# Multi-Source-AI-News-Aggregator-Categorization-System
This project automatically collects technology news from multiple sources, categorizes articles using AI, generates concise summaries, removes duplicates, and stores the results in a Supabase database for further analysis or publishing.
- Automated RSS ingestion
- Multi-source aggregation
- AI-powered categorization
- AI-generated summaries
- Duplicate detection
- Supabase database storage
- Fully automated workflow
 
# Tech stack
- n8n
- OpenRouter
- LLM (GPT / Claude / Gemini)
- Supabase
- RSS Feeds
- JavaScript
